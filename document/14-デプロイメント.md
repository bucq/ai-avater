## 14.3 環境別設定

### 開発環境

```bash
# frontend/.env.development
VITE_API_URL=http://localhost:8000

# backend/.env.development
ENV=development
DEBUG=true
```

### 本番環境

```bash
# frontend/.env.production
VITE_API_URL=https://api.yourdomain.com

# backend/.env.production
ENV=production
DEBUG=false
```
