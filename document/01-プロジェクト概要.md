---

## 1.1 目的

AI応答に連動してアニメーションするアバターを表示し、自然な対話体験を提供するシステム

## 1.2 スコープ (Phase 1)

- 3Dアバター表示とアニメーション制御
- AIチャット機能
- リップシンク
- 感情表現
- アイドルモーション

## 1.3 開発方針

### Phase 1 の目標
1. **アバター表示**: [[04-アバター表示機能|VRMモデル]]の3D表示
2. **アニメーション**: [[05-アニメーション機能|事前定義モーション]]の再生
3. **AI対話**: [[06-AI対話機能|OpenAI連携]]による自然な会話
4. **音声同期**: [[05-アニメーション機能#リップシンク|リップシンク]]による口の動き

### 開発期間
- **期間**: 9週間
- **詳細**: [[12-開発スケジュール|開発スケジュール]]を参照

### 技術選択理由
- **フロントエンド**: Vite + React → Three.jsとの相性、高速開発
- **3D**: Three.js + VRM → アニメ調アバターに最適
- **バックエンド**: FastAPI → Python AIライブラリとの親和性
- 詳細は [[03-技術スタック|技術スタック]]を参照

## 1.4 制約事項

### Phase 1 の制限
```
✗ 音声入力未対応
✗ マルチユーザー未対応
✗ 会話履歴の永続化なし
✗ カスタムアバター変更不可
✗ カメラ入力不要（事前定義アニメーション）
```

### 技術的制約
```
- WebGL 2.0 対応ブラウザ必須
- VRMモデルサイズ: 50MB以下
- 同時アニメーション数: 最大4つまで
- 音声ファイルサイズ: 5MB以下
```

詳細は [[08-非機能要件|非機能要件]]を参照
